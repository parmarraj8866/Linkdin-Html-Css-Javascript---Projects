What is HTML?

HTML (HyperText Markup Language) is the standard language used to create the structure of web pages. It defines elements like headings, paragraphs, links, images, etc.

What is CSS?

CSS (Cascading Style Sheets) is used to style HTML elements. It controls colors, fonts, layouts, spacing, and responsiveness of a webpage.

What is JavaScript?

JavaScript is a programming language that allows you to add interactivity, animations, and dynamic content to websites. It runs in the browser and is essential for modern web development.
